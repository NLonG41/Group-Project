# Thư mục Public

Đặt file logo của trường vào đây.

## Tên file logo:
- `logo.png` - Logo chính (khuyến nghị)
- Hoặc `logo.jpg`, `logo.svg`

## Cách sử dụng:
1. Đặt file logo vào thư mục này với tên `logo.png` (hoặc `logo.jpg`, `logo.svg`)
2. Component Logo sẽ tự động load file này
3. Nếu không có file, sẽ hiển thị text "USTH" làm fallback

## Định dạng khuyến nghị:
- PNG với nền trong suốt
- Kích thước: 200x200px trở lên
- Độ phân giải cao để hiển thị rõ trên mọi màn hình
